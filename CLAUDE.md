# UI Design Rules

本项目已经确定 Dashboard 首页设计。

以后所有页面必须严格遵守首页风格。

禁止：

- 修改整体布局
- 修改颜色体系
- 修改圆角
- 修改字体
- 修改Sidebar
- 修改TopBar
- 修改Card样式

所有新增页面必须继承首页设计。

设计语言：

- 深色主题
- 科技感
- 游戏风格
- 大量Card布局
- shadcn/ui
- TailwindCSS
- 柔和阴影
- 圆角12~16px
- Hover动画统一
- 使用Heroicons/lucide-react

如果需要新增组件，
优先复用已有组件，
禁止重新设计Button。

所有页面必须看起来像同一个产品。
